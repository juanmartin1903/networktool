Applikationen kontrollerar den angivna IP-adressen och portnumret och returnerar om den är giltig eller inte.
Du kan också välja en rapport med alla testade IP-adresser och portar.
Visual Studio Code-applikationen hjälper till att testa koden utan att kompilera den. Du behöver bara lägga till C/C+ och Python-tillägg.
Jag har tittat på några långa Python-instruktionsvideor under den här perioden, men den sista var den som visade mig hur man förenklar med en felsökare för att se vad varje rad i koden gör. Med C måste man kompilera varje gång man vill se hur programmet fungerar. Man kan också kompilera som en exe från Python, men det är annorlunda, det packar koden, tolken och biblioteken.
Python blir mycket enklare att koda, till exempel för att kunna kontrollera IP importerar man "regex". I C måste man använda sscanf, flera bibliotek och limiters. TAB-tangenten hjälper också till med förslag, vet inte om det är ett nytt tillägg som följde med IA men det är en stark hjälp för att kunna koda snabbare utan att googla efter alla kommandon.
Det som jag förstå är att C är mycket mer effektiv med jobbigare eller tunga procceser. 
I slutet, för att kunna jobba med enkla applikationer skulle jag välja Python medan om det behövs mer , skulle jag välja C.
Vad jag förstår är att C är mycket effektivare med mer krävande funktioner.
